Data type /
varchar - строка, которая может быть разной длины, используется для хранения текста, например, имен или адресов.
char - строка фиксированной длины
int - целое число
decimal - десятичное число

time - время

--------------------------
Constraints - ограничения

NOT NULL - гарантирует, что в поле всегда будет значение
UNIQUE - обеспечивает, что значения в поле будут уникальными
PRIMARY KEY - уникальный идентификатор для каждой строки в таблице
FOREIGN KEY - связывает поле с другим столбцом в другой таблице
CHECK -  проверяет, что значение в поле соответствует заданному условию
DEFAULT -  устанавливает значение по умолчанию для поля
CREATE INDEX - создает индекс
