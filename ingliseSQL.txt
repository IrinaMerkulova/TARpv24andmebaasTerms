Constraints

NOT NULL - Столбец не может быть пустым (NULL).
UNIQUE - Значения в столбце должны быть уникальными.
PRIMARY KEY - Столбец (или группа столбцов), который идентифицирует каждую строку таблицы уникально.
FOREIGN KEY - Столбец ссылается на первичный ключ в другой таблице.
CHECK - Ограничение, которое проверяет, что значения в столбце соответствуют указанному условию.
DEFAULT - Устанавливает значение по умолчанию для столбца, если оно не указано.
CREATE INDEX - Создает индекс для ускорения поиска данных в таблице.
