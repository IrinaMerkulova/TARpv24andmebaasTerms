SQL - (Structured Query Language) Язык структурированных запросов
DDL - (Data Definition Language) Язык описания данных
DML - (Data Manipulation Language) Язык управления (манипулирования) данными
Database - База данных
Database management system - Система управления базой данных
Related table - Связаная таблица
Relational database - Реляционная база данных
Relationship - Отношения
 
Primary Key - Первичный ключ
Foreign Key - Вторичный ключ
One-to-many relationship - Отношения "один ко многим"
One-to-one relationship - Отношения "один к одному"
Query - Запрос
Report - Отчет
Form - Форма

Data type - Типы данных
varchar - Строка, которая может быть разной длины
char - Строка фиксированной длины
int - Целое число
decimal - Десятичное число
text - Текст
bit - Число которое можеть быть только 0 или 1
bool - Логическое значение False/True (0/1)
date - Дата
time - Время
blob - (Binary Large Object) Тип данных для хранение больших обьемов двоичных данных
