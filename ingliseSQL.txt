SQL - Structured Query Language
DDL - Data Definition Language
DML - Data Manipulation Language
Database - Организованный набор данных, хранящихся в таблицах.
Database management system - Программное обеспечение для управления базами данных.
Related table - Таблица, связанная с другой через ключи.
Relational database - База данных, где данные хранятся в таблицах, связанных между собой.
Relationship - Связь между таблицами в базе данных.
 
Primary Key - Уникальный идентификатор записи в таблице.
Foreign Key - Ключ, который связывает таблицы, ссылаясь на первичный ключ другой таблицы.
One-to-many relationship - Связь, где одна запись в таблице может быть связана с несколькими записями в другой.
One-to-one relationship - Связь, где одна запись в таблице связана с одной записью в другой.
Query - Запрос на извлечение данных из базы данных.
Report - Отчёт — форматированное представление данных.
Form - Интерфейс для ввода и редактирования данных.
Data type - тип данных
 varchar - строка переменной длины/ колво символов
char - строка фиксированной длины
 int - оператор целых чисел
 decimal - оператор дробных чисел
 text - текст
 bit - двоичный оператор
 bool - логический тип, true/false
 date - оператор даты
 time - оператор времени
 blob - двоичные данные

FOREIGN KEY - вторичный ключ
CHECK - проверка
DEFAULT - по умолчанию
